#  Player Profile Website

This is a simple football player profile website I built using **HTML, CSS, and a bit of JavaScript**.
It shows my player stats, attributes, and highlight videos in a clean and modern layout.

The goal of this project was to practice front-end development and create something that actually represents me as a player.

---

##  Pages in the Project

### 1. **index.html – Main Profile Page**

This is the main page of the website.

It includes:

* Player information (name, age, height, position)
* Overall stats like appearances, goals, assists, and rating
* Season selector to view stats from different seasons
* Animated UI (hover effects, particles, scroll animations)
* Navigation bar to switch between pages

The stats update automatically depending on the season you select.

---

### 2. **attributes.html – Player Attributes**

This page shows my football attributes like:

* Speed
* Dribbling
* Shooting
* Passing
* Stamina
* Defense
* Physicality
* Workrate

It has:

* A different design style from the main page
* Background image
* Grid layout for attributes
* Simple hover effects

---

### 3. **skills.html – Video Highlights**

This page shows my football highlights using embedded YouTube videos.

Features:

* Two highlight videos (Skills & Goals 1 and 2)
* Buttons to switch between videos
* Simple and clean layout
* Background image and fade-in animation

## How to Run the Project

1. Download or clone the project.
2. Open `index.html` in your browser.
3. Use the navigation bar to move between pages.

OR visit:
https://daviskennedy10.github.io/ODJ-Football-Page/

No installation or setup needed.

##  Notes

* The stats are hard-coded using JavaScript.
* Videos are embedded from YouTube.
* Images need to be in the same folder as the HTML files.
